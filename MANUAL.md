### Teleoperator Manual
#### Controls
1. Left Joystick - Control antennas (up/down)
2. Right Joystick - Control head (look around)
3. Action Buttons - Trigger animations
4. Head Button - Phone orientation to head orientation
---
#### When to Act
| Situation | Action |
|-----------|--------|
| Greeting | WAVE |
| Yes answer | WAVE |
| No answer | SHAKE HEAD NO |
| Surprised reaction | SURPRISED |
| Happy reaction | HAPPY |
| Sad reaction | SAD |
| Laughing | LAUGH |
